# File Upload Bypass Wordlist

A comprehensive wordlist designed to test and identify file upload vulnerabilities in web applications through various bypass techniques.

## Overview

This wordlist contains a curated collection of file extensions and techniques used to bypass file upload filters, including:
- Double extensions
- Reverse double extensions
- Character injections

## Purpose

This resource is designed for **security professionals, pentesters, and bug bounty hunters** who are conducting security assessments and vulnerability research on web applications.

## Usage

Use this wordlist with your preferred security testing tools or manual testing approaches to identify file upload vulnerabilities. My recomendations: Burp intruder and ZAP fuzzer

## Disclaimer

This wordlist is intended for **authorized security testing only**. Unauthorized access to computer systems is illegal. Always obtain proper authorization before testing.
